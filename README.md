# Division Adventure

Application web mobile-first pour gérer un héros de jeu de rôle dans un style inspiré de The Division.

## Fonctionnalités

- Écran d’accueil responsive
- Connexion par nom de héros
- Création d’un nouveau héros
- Tableau de bord du héros avec statut, inventaire et mission
- Stockage local des héros et session dans le navigateur

## Installation

1. Installez les dépendances : `npm install`
2. Lancez le serveur local : `npm start`
3. Ouvrez `http://localhost:3000` dans un navigateur moderne.

## Utilisation

1. Saisissez le nom de l’agent et son mot de passe pour vous connecter.
2. Cliquez sur "Créer un nouvel agent" pour enregistrer un nouveau profil.
3. Remplissez les informations demandées, puis validez.
4. Le nouvel agent est enregistré dans la base SQLite du serveur.
